¡Claro! Aquí tienes la resolución paso a paso de los ejercicios de tu tarea, evaluando las funciones en los puntos solicitados y determinando su comportamiento cuando tienden al infinito.
------------------------------
Ejercicio 1: $f(x) = x^2 + 2$
Evaluación para $x \to 1, 2, 3$:

* Si $x = 1$: $f(1) = (1)^2 + 2 = 1 + 2 = \mathbf{3}$
* Si $x = 2$: $f(2) = (2)^2 + 2 = 4 + 2 = \mathbf{6}$
* Si $x = 3$: $f(3) = (3)^2 + 2 = 9 + 2 = \mathbf{11}$

Límite al infinito:
A medida que $x$ crece sin control, el término $x^2$ crece mucho más rápido.
$$\lim_{x \to \infty} (x^2 + 2) = \mathbf{\infty}$$ (La función diverge al infinito).
------------------------------
Ejercicio 2: $f(n) = n^2 + 1$
Evaluación para $n \to 1, 2, 3$:

* Si $n = 1$: $f(1) = (1)^2 + 1 = 1 + 1 = \mathbf{2}$
* Si $n = 2$: $f(2) = (2)^2 + 1 = 4 + 1 = \mathbf{5}$
* Si $n = 3$: $f(3) = (3)^2 + 1 = 9 + 1 = \mathbf{10}$

Límite al infinito:
Igual que el anterior, al ser una potencia positiva:
$$\lim_{n \to \infty} (n^2 + 1) = \mathbf{\infty}$$ 
------------------------------
Ejercicio 3: $a_n = \frac{n^2}{2}$
En este caso, solo se pide el comportamiento cuando $n$ tiende a infinito.
Límite al infinito:
Como el numerador $n^2$ crece infinitamente y se divide por una constante, el resultado sigue creciendo.
$$\lim_{n \to \infty} \frac{n^2}{2} = \mathbf{\infty}$$ 
------------------------------
Ejercicio 4: $a_n = \left[ \frac{n}{2} \right]^2$
Límite al infinito:
Al elevar al cuadrado una fracción donde el numerador crece infinitamente, el resultado también será infinito.
$$\lim_{n \to \infty} \left[ \frac{n}{2} \right]^2 = \mathbf{\infty}$$ 
------------------------------
Resumen: En todos los casos, como el término de mayor grado es positivo ($x^2$ o $n^2$), las funciones y sucesiones divergen a infinito cuando la variable crece.
